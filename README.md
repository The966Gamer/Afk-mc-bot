# Minecraft AFK Bot 🤖

A simple Mineflayer bot that joins a Minecraft server, chats randomly, and avoids AFK kicks.

## 🚀 Features
- Auto reconnect
- Random chat messages
- Basic anti-AFK movement
- Simple chat responses
